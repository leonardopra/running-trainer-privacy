1. Introduzione

Benvenuto in Running Trainer. La presente informativa descrive come raccogliamo, utilizziamo e proteggiamo i tuoi dati personali quando utilizzi la nostra applicazione.

Running Trainer non richiede account, login o registrazione. Tutti i dati sono salvati localmente sul tuo dispositivo e non vengono mai trasmessi a server esterni, salvo quanto indicato nella sezione Claude AI.

2. Dati raccolti e memorizzati

L'app raccoglie e memorizza localmente sul tuo dispositivo le seguenti informazioni:

  • Dati del profilo: nome, età, peso e altezza inseriti volontariamente per personalizzare i piani di allenamento.
  • Dati di allenamento: piani di allenamento, sessioni completate, distanza, ritmo, durata e obiettivi.
  • Notifiche: i promemoria di allenamento sono pianificati localmente sul dispositivo. Nessun dato viene inviato a server esterni.

3. Claude AI (opzionale)

L'app offre un'integrazione opzionale con l'API di Claude (Anthropic) per generare descrizioni di coaching e feedback post-allenamento.

  • L'utilizzo di questa funzione richiede l'inserimento di una API key personale di Anthropic.
  • Se fornita, i dati degli allenamenti vengono inviati alle API di Anthropic per generare i contenuti.
  • Questa funzione è completamente opzionale — l'app funziona interamente offline senza API key.
  • Per informazioni su come Anthropic gestisce i dati, consulta la Privacy Policy di Anthropic.

⚠️ Avviso di sicurezza: la API key viene salvata in chiaro nelle preferenze locali del dispositivo. Non utilizzare questa app su dispositivi condivisi o con root se desideri mantenere segreta la tua chiave.

4. Nessun tracciamento

Running Trainer non include analytics, crash reporting o tracciamento dell'utilizzo. Non sono presenti SDK di terze parti con funzionalità di tracking.

5. Sicurezza dei dati

I dati del profilo (nome, età, peso, altezza) sono memorizzati localmente sul dispositivo. I piani di allenamento, i log degli allenamenti e le informazioni del profilo non lasciano mai il dispositivo, a meno che tu non li condivida esplicitamente o non utilizzi la funzione Claude AI.

6. Dati di minori

Running Trainer non è destinata a bambini di età inferiore ai 13 anni. Non raccogliamo consapevolmente dati da minori. Se sei un genitore o tutore e ritieni che tuo figlio abbia fornito dati tramite l'app, ti invitiamo a contattarci.

7. Eliminazione dei dati

Puoi eliminare tutti i dati dell'app in qualsiasi momento tramite Impostazioni → Reset All Data. La disinstallazione dell'app rimuove tutti i dati salvati localmente.

8. Modifiche alla presente informativa

Ci riserviamo il diritto di aggiornare questa informativa. In caso di modifiche sostanziali, ti informeremo tramite un avviso nell'app. La data di ultimo aggiornamento è indicata in cima a questa pagina.

1. Introduction

Welcome to Running Trainer. This privacy policy describes how we collect, use, and protect your personal data when you use our application.

Running Trainer requires no account, login, or registration. All data is stored locally on your device and never transmitted to external servers, except as described in the Claude AI section.

2. Data Collected and Stored

The app collects and stores the following information locally on your device:

  • Profile data: name, age, weight, and height voluntarily entered to personalize training plans.
  • Training data: training plans, completed sessions, distance, pace, duration, and goals.
  • Notifications: workout reminders are scheduled locally on your device. No data is sent to external servers.

3. Claude AI (Optional)

The app offers an optional integration with Claude's API (Anthropic) to generate coaching descriptions and post-workout feedback.

  • Using this feature requires providing a personal Anthropic API key.
  • If provided, workout data is sent to Anthropic's API to generate content.
  • This feature is entirely optional — the app works fully offline without an API key.
  • For information on how Anthropic handles data, refer to Anthropic's Privacy Policy.

⚠️ Security notice: the API key is stored in plain text in local device preferences. Do not use this app on a shared or rooted device if you care about keeping your key secret.

4. No Tracking

Running Trainer does not include analytics, crash reporting, or usage tracking. No third-party SDKs with tracking capabilities are included.

5. Data Security

Profile data (name, age, weight, height) is stored locally on your device. Training plans, workout logs, and personal profile information never leave your device unless you explicitly share them or use the Claude AI feature.

6. Children's Data

Running Trainer is not intended for children under the age of 13. We do not knowingly collect data from minors. If you are a parent or guardian and believe your child has provided data through the app, please contact us.

7. Data Deletion

You can delete all app data at any time via Settings → Reset All Data. Uninstalling the app removes all locally stored data.

8. Changes to This Policy

We reserve the right to update this privacy policy. In case of substantial changes, we will notify you via an in-app notice. The last update date is shown at the top of this page.
